Projetado para todos, em qualquer lugar.

O Bootstrap torna o desenvolvimento web front-end mais rápido e fácil. É feito para pessoas de todos os níveis, qualquer tipo de dispositivo, e projetos de todos os tamanhos.

Pré-processadores

Como complemento ao vanilla CSS, Bootstrap oferece suporte aos dois pré-processadores de CSS mais pupulares, Less e Sass.

Um framework, qualquer dispositivo.

Bootstrap escala seu projeto fácil e eficientemente com uma base de código, desde telefones a tables e desktops.

Documentação detalhada

Com bootstrap, você possui uma documentação extensa e bonita com centenas de exemplos demonstrativos, trechos de código, e mais.

Bootstrap é open source. É hospedado, desenvolvido e mantido no GitHub.
